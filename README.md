# ناجز — Najez

App for collecting daily tasks and consolidating them into achievements.

تطبيق ويب بسيط لمتابعة مهام العمل اليومية وتحويلها إلى لوحة إنجازات سنوية.

## الصفحات

- **اليوم** — سؤال واحد: «إيش اشتغلت عليه اليوم؟». اكتب المهام التي أنجزتها وستُحفظ تلقائيًا.
- **الإنجازات** — لوحة أرقام لسنة كاملة:
  - إجمالي المهام، أيام العمل، متوسط المهام في اليوم، أطول سلسلة أيام متتالية.
  - رسم بياني بعدد المهام شهريًا (مع جدول أرقام).
  - خريطة نشاط للسنة على طريقة خريطة المساهمات.
  - سجل كامل للمهام مع إمكانية الحذف.

## التشغيل

التطبيق ثابت بالكامل (HTML/CSS/JS) بدون أي اعتماديات أو خطوة بناء:

```bash
# افتح index.html مباشرة في المتصفح، أو شغّل خادمًا محليًا:
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

يصلح للنشر مباشرة على GitHub Pages أو أي استضافة ملفات ثابتة.

## التخزين

تُحفظ البيانات محليًا في متصفحك عبر `localStorage` — لا يوجد خادم ولا تغادر بياناتك جهازك.

في صفحة الإنجازات يوجد زر «جرّب ببيانات تجريبية» لتعبئة بيانات وهمية لمعاينة اللوحة، وزر لحذفها لاحقًا.
